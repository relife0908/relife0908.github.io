Ubantu下运行Clash For Windows

发表于2023-08-18|更新于2025-08-31|教程

偶然发现Clash For Windows并不是只能在Windows下使用，在Ubantu下同样可以使用(所以你为什么要叫windows?)，命令行的方式也可以运行clash，但图形界面更易上手

后续得知的原因：windows指的是图形界面，而不是windows系统

以下为简单的使用步骤：

1.下载地址

https://github.com/Fndroid/clash_for_windows_pkg/releases
amd64版本请选择Clash.for.Windows-0.20.31-x64-linux.tar.gz

25年更新：
已经失效，这里找到替代的中文版本
https://github.com/Z-Siqi/Clash-for-Windows_Chinese

2.使用教程

解压文件，可以看见文件cfw，即为我们需要运行的应用

打开终端，运行文件，/home/（文件位置）/cfw

打开成功，此时请勿关闭终端，关闭终端会导致程序直接结束运行

正常操作即可
